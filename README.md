## Setup
```
git clone https://github.com/ferater/Single-Product-Bag.git
```
```
cd Single-Product-Bag
```
```
yarn 
#or 
npm install
```
```
yarn dev 
#or 
npm run dev
```
```
visit http://localhost:5173/Single-Product-Bag/
```
